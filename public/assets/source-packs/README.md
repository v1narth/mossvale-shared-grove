# Source Asset Packs

Original downloaded asset packs are unpacked here so the game can load assets directly and we can browse them later.

- `Ultimate Nature Pack - Jun 2019/` - trees, rocks, plants, grass, logs, OBJ/FBX/Blender source files.
- `Ultimate Monsters/` - blob, flying, and large creature models with glTF/OBJ/FBX/Blender source files.
- `KayKit_Adventurers_2.0_FREE/` - adventurer characters, gear, textures, and animations.
- `Medieval Village MegaKit[Standard]/` - modular medieval buildings, props, textures, glTF/OBJ/FBX source files.
- `kenney_fantasy-ui-borders/` - fantasy UI border PNG and vector artwork.
